# D&D-attack
Приветствую,я начинающий програмист на языке Javascript,моя 1 программа которую выкладываю в открытый доступ.
Программа про D&D механику боя,когда у врага определённое КБ(класс-брони) и ты должен выбрость больше или равное кол-во КБ у пративника на классическом d20 кубике,с крит поподанием и промохом.
